Design

I prefere to develope Outside In style. Start with fast prototyping with Adobe Fireworks then HTML/CSS with TwitterBootstrap and last but not least Ruby on Rails.

HTML/CSS

I'm using for views development CSS with Sass or Less and TwitterBootstrap. For javascript jQuery. If this is bigger project I prefer to use OOP views with Erector.

Ruby

I love to work with Ruby and Ruby on Rails. If the project is really small application/micro page I use Sinatra with support Padrino.

Testing and BDD, TDD

I think this should be the standard on every ROR project. Development with Cucumber and RSpec is perfect solution for BDD or TDD. But on projects that I was involved this was newer the request.

I'm trying some touch with Cucumber/Rspec on my own projects. This is my priority for now to do better development.
